# ✊✋✌️ Stone Paper Scissors Game (C++)

A simple **Stone – Paper – Scissors** console game built using **C++**.
The player competes against the computer for several rounds, and the program determines the winner of each round and the final game result.

---

## 🎮 Game Description

The player chooses one of the following options:

1. **Stone**
2. **Paper**
3. **Scissors**

The computer randomly selects its choice.

### Rules

* **Stone beats Scissors**
* **Scissors beats Paper**
* **Paper beats Stone**
* If both choose the same → **Draw**

## 🧩 Main Components

### Enums

* `enGameChoice` → Represents the player's or computer's choice
* `enWinner` → Represents the round or game winner

### Structs

* `stRoundInfo` → Stores information about each round
* `stGameResults` → Stores final game statistics

### Important Functions

* `RandomNumber()` → Generates random numbers
* `GetComputerChoice()` → Gets computer move
* `WhoWonTheRound()` → Determines round winner
* `WhoWonTheGame()` → Determines final winner
* `PlayGame()` → Runs the game rounds
* `StartGame()` → Starts and manages the game
